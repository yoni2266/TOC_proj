1. Install XAMPP Control Panel.
2. Open the Panel and start the module "Apache" which can open a root here.
3. Make sure the file "TOC_proj.php" and "proj.php" are at the specified folder "\xampp\htdocs\".
4. Also, don't forget to download the data "data.json" and put it at the location "\xampp\htdocs\".
5. Click a browser and type "localhost/TOP_proj.php" to execute the program.
